# Registro de Alterações (Changelog)

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-11-10

### Adicionado
- **Licenciamento Legal**
  - Adicionado arquivo LICENSE com licença MIT completa
  - Definições claras de direitos de uso e redistribuição

- **Código Funcional Completo**
  - Monitor de filamento ESP32 com interface web
  - Sistema de esteira transportadora Arduino
  - Estação de controle de qualidade Raspberry Pi com OpenCV

- **Modelos 3D Paramétricos**
  - Suporte para carretel de filamento (OpenSCAD)
  - Rola para esteira transportadora (OpenSCAD)
  - Documentação completa de impressão

- **Estrutura de Projeto Profissional**
  - Organização modular por hardware
  - Documentação técnica detalhada
  - Guias de contribuição para desenvolvedores

- **Configuração de Ambiente**
  - Script de instalação automatizada (setup-3dpot.sh)
  - Dependências organizadas por tipo de hardware
  - Configuração de ambiente de desenvolvimento

- **Roadmap de Desenvolvimento**
  - 5 issues principais criadas no GitHub
  - Planejamento de CI/CD e testes automatizados
  - Estrutura para crescimento futuro

### Melhorado
- **Estrutura do Repositório**
  - Reorganização de arquivos por funcionalidade
  - Limpeza de arquivos desnecessários
  - Otimização de .gitignore específico para maker projects

- **Configuração de Desenvolvimento**
  - pyproject.toml modular e otimizado
  - Dependências separadas por tipo de projeto
  - Configuração de ferramentas de qualidade de código

- **Documentação**
  - Código bem comentado e documentado
  - Guias de instalação específicos por hardware
  - Exemplos de uso e troubleshooting

### Crítico
- **Base Legal**: Adicionada licença MIT para permitir contribuições da comunidade
- **Funcionalidade**: Todos os projetos têm código funcional e testável
- **Organização**: Estrutura profissional adequada para crescimento
